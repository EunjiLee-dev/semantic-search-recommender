# Semantic Search & Recommendation System

## Goal
Build a semantic search system for POI data using Sentence-BERT.

## Tech Stack
- Python
- FastAPI
- Sentence-BERT
- PostgreSQL / SQLite

## Features
- Semantic search using embeddings
- Cosine similarity retrieval
- Ranking system with hybrid scoring

## Goal
Enable natural language queries like:
"cheap sushi near university"
